## 这份资料讲了什么

2026-06-22 召开的项目启动会，涵盖项目背景、技术选型、MVP 范围、里程碑和风险五个议题。会议确定了「吾家有老」社区养老平台 MVP 的核心功能范围和 8 周交付计划。

## 为什么保留

这是项目的**第一份正式记录**，包含所有关键技术决策和团队共识，是后续开发的基准线。

## 关键结论

1. **技术栈确定**：NestJS + MySQL 8 + Redis + 微信原生小程序 + Docker Compose
2. **MVP 四大核心**：打卡、紧急呼叫、状态查看、服务工单
3. **8 周里程碑**：M0 脚手架 → M1 打卡 → M2 家属端 → M3 工单联调
4. **关键风险**：微信审核资质、数据加密合规、人力紧张
